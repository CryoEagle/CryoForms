Class: CryoInput

CryoInput(defaultValue, value, onChange, label, placeholder, name, description, type, rows, rules, autoComplete, inputProps, errorMessProps, descriptionProps, onNotValidChange, onValidChange, fileInputRightSideText, disallowFormGroup, quillModules)

new CryoInput(defaultValue, value, onChange, label, placeholder, name, description, type, rows, rules, autoComplete, inputProps, errorMessProps, descriptionProps, onNotValidChange, onValidChange, fileInputRightSideText, disallowFormGroup, quillModules)

Creates input, but very fancy with lot of cool stuff there, you can use very advanced inputs

Parameters:
Name Type Description
defaultValue string

Default value means that input will save value to state only at start, lot of times it's enough.
value string

Operates with state whole time.
onChange function

Is called when input changed.
label string

Label is something like title, visible even when input is filled.
placeholder string

Classic placeholder like in HTML.
name string

Very important and will be used as object property while saving to JSON.
description string

Small text for user, can be filled with some instruction or whatever.
type string

This input don't support all types, some of the types must be set in rules, allowed types are text, password, textarea, richtext, file
rows number

Is used when input type is textarea
rules Array.<InputRules>

For example rules={[{required: true, errorMessage: 'This input muset be filled'}, {...}]}
autoComplete string

You can set autoComplete='off'.
inputProps props

Custom props of input.
errorMessProps props

Custom props of error message.
descriptionProps props

Custom props of description component.
onNotValidChange function

Function called when change is not valid.
onValidChange function

Function called when change is valid.
fileInputRightSideText string

Text in file input on right side.
disallowFormGroup bool

Removes class with bottom margin.
quillModules quillProps

Set custom props to quill richtext editor
